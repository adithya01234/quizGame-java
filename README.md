# 🧠 Quiz Game Application (Java)

<p align="center">
  <b>A console-based Quiz Game built using Core Java and OOP principles</b>
</p>

---

## 📌 Overview

The **Quiz Game Application** is a Java-based console program designed to simulate a multiple-choice quiz environment.
It demonstrates core programming concepts such as **Object-Oriented Programming (OOP), modular design, and reusable utility functions**.

The application presents questions to the user, evaluates responses, and displays the final score.

---

## ✨ Key Features

* 📝 Multiple-choice questions (MCQs)
* ⚡ Instant answer validation
* 📊 Final score calculation
* 🔄 Modular and scalable code structure
* 🧩 Utility class for reusable functions (input handling, validations, etc.)

---

## 🛠️ Tech Stack

* **Language:** Java
* **Concepts:** OOP (Encapsulation, Classes, Objects)
* **Tools:** VS Code / IntelliJ IDEA / Eclipse

## 🧩 Utility Class (Important)

The `util` package contains helper methods used across the application.

Example responsibilities:

* Handling user input
* Validating answers
* Reusable common functions

This improves:

* Code reusability
* Clean architecture
* Separation of concerns
